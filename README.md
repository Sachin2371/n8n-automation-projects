# 🔔 NSUT Notice Bot

Automatically fetches notices from NSUT website 
and sends them to a Telegram channel every 30 minutes!

## 🚀 Live Channel
👉 [t.me/nsut_notices_official](https://t.me/nsut_notices_official)

## ⚙️ How It Works
NSUT Website (every 30 min)
↓
n8n checks for new notices
↓
New notice found?
YES → Send to Telegram Channel
NO  → Skip (no spam!)

## 🛠️ Tech Stack
- **n8n** — Workflow automation
- **HTTP Request** — Website fetching
- **JavaScript** — HTML parsing
- **Telegram Bot API** — Notifications
